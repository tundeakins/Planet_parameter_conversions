# Planet Tools
Useful functions to perform everyday conversions and calculation of quantities in exoplanetary science.
The module Convert_params contains:
\item 'a_r' - convert period to scaled semi-major axis.
\item 'a_r_to_rho_star' - Compute transit derived stellar density from the planet period and scaled semi major axis
\item 'impact_parameter'- convert inclination to impact parameter b
 
\item 'inclination' - convert impact parameter b to inclination in degrees.
\item 'kipping_ld' - Re-parameterize quadratic limb darkening parameters $u_{1}$ and $u_{2}$ according to Kipping (2013)
\item 'prot'- convert stellar rotation velocity vsini in km/s to rotation period in days.
\item 'rho_to_a_r'- convert stellar density to semi-major axis of planet with a particular period
\item 'vsini'- convert stellar rotation period to vsini in km/s.
